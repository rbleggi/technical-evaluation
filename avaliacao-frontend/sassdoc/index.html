    @function strip-hyphen($string) { ... }

    Description

    Remove qualquer hífen de uma string e a concatena.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $string

    String com hífen

    		String— none

    Returns

    String —

    String sem hífen

    Resultado da função strip-hyphen:

  background-image: url(strip-hyphen('string-com-hifen'));

    font-import

    @mixin font-import($family, $weight, $style, $file) { ... }

    Description

    Define uma @fonte-face.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $family

    Família da fonte a ser importada

    		String— none
    $weight

    Peso da fonte a ser importada

    		Number— none
    $style

    Estilo da fonte a ser importada

    		String— none
    $file

    A string deve corresponder ao nome do arquivo a ser importado

    		String— none

    Example

    Resultado do import do mixin font-import:

    @include font-import('Roboto', 700, normal, roboto-bold-normal);

@font-face {
  font-family: "Roboto";
  font-weight: 700;
  font-style: normal;
  src: url("../assets/roboto-bold-normal.eot");
  src: url("../assets/roboto-bold-normal.eot?#iefix") format("embedded-opentype"),
      url("../assets/roboto-bold-normal.woff2") format("woff2"),
      url("../assets/roboto-bold-normal.woff") format("woff"),
      url("../assets/roboto-bold-normal.ttf") format("truetype"),
      url("../assets/roboto-bold-normal.svg#robotoboldnormal") format("svg");
}
